# Restaurant photo drop-in folder

Real JPG/PNG restaurant photos are not uploaded to the repository yet.

When the files are added here, `src/main.js` will automatically use them in the website if they match these filenames:

- `hero-tacos-drinks.jpg` — homepage hero background
- `order-burger.jpg` — Order Online panel
- `reserve-venue.jpg` — Reserve a Table panel
- `special-tacos.jpg` — Taco Tuesday card
- `special-burger.jpg` — Wednesday Burger Night card
- `special-margarita.jpg` — Happy Hour card
- `special-kids.jpg` — After School / kids card
- `feature-tacos.jpg` — Featured tacos card
- `feature-burger.jpg` — Featured burgers card
- `feature-grill.jpg` — Featured grill card
- `feature-mexican.jpg` — Featured Mexican favourites card
- `drink-margarita.jpg` — Margaritas card
- `drink-cocktail.jpg` — Cocktails card
- `drink-mocktail.jpg` — Mocktails card
- `gallery-tacos.jpg` — Gallery tacos card
- `gallery-burgers.jpg` — Gallery burgers card
- `gallery-ribs.jpg` — Gallery ribs/grill card
- `gallery-mocktails.jpg` — Gallery mocktails card
- `gallery-kids.jpg` — Gallery kids meals card
- `gallery-venue.jpg` — Gallery restaurant vibe card

Use relative paths only, for example: `./assets/photos/hero-tacos-drinks.jpg`.
