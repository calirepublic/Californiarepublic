# California Republic

A complete, responsive homepage for **California Republic St Ives** that can be previewed directly in the browser.

## Homepage sections

- Hero banner with primary online-ordering and reservation calls to action
- Online ordering cards for tacos, burgers, and grill favourites
- Table reservation request form
- Happy Hour, Taco Tuesday, and Burger Night promotions
- Kids Menu section for family dining
- Gallery section with visual placeholders ready for real restaurant photography
- Contact section for address, inquiry, booking links, menu PDFs, app planning, and online-ordering setup

## Live preview

This project is configured for GitHub Pages. After this branch is merged to `main`, GitHub Actions will publish the static homepage and expose the live URL from the workflow environment output.

You can also create an immediate browser preview locally:

```bash
npm run preview
```

Then open <http://localhost:4173>.

## Local development

```bash
npm start
```

Then open <http://localhost:4173>.

## Validation

```bash
npm run check
```
